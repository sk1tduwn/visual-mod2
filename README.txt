This folder should contain the Gradle wrapper JAR: gradle-wrapper.jar

The repository contains `gradlew` and `gradlew.bat` scripts and `gradle-wrapper.properties`.
To use the wrapper without a local Gradle installation, place the `gradle-wrapper.jar` into this folder.

If you have Gradle installed locally, you can generate the wrapper jar by running:

  gradle wrapper

or download a matching `gradle-wrapper.jar` from a trusted source.
